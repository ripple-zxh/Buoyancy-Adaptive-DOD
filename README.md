# Buoyancy-Adaptive DOD Surrogate for Transient Rod-Bundle Thermal-Hydraulics

This repository accompanies the manuscript:

**A Buoyancy-Adaptive Reduced-Order Surrogate for Heated Rod-Bundle Transients Using Gauge-Consistent Deep Orthogonal Decomposition**

The repository will provide the implementation used in the paper, including:

* the buoyancy-adaptive Deep Orthogonal Decomposition (DOD) reduced-order model;
* gauge-alignment and Procrustes-based basis-consistency procedures;
* POD and clustered-POD baseline comparisons;
* ablation studies for DOD, POD, and coefficient-regression models;
* scripts for reproducing the main reconstruction-error and modal-cost analyses.

The source code will be released after publication of the paper.

Due to the large size of the geometry, mesh, and CFD snapshot files, these data are not hosted directly in this GitHub repository. They may be made available upon reasonable request. Please contact the author through GitHub or by email for access to the large files required for full reproduction.
